# Data Model for Project Frameworks

This repository has been intentionally stripped down.

The original working files, sample corpora, notebooks, PDFs, spreadsheets, images, graph exports, and generated outputs have been removed from this public copy. What remains is the navigational skeleton for a project and portfolio framework workbench.

## Structure

- `P3M-content-first-framework/` - modular portfolio and project-management service framework.
- `Project-frameworks-by-using-NLP-in-Orange-Datamining/` - Orange Data Mining workflow area.
- `Project-frameworks-by-using-NLP-with-Python-libraries/` - Python and Jupyter NLP workflow area.
- `Project frameworks by starting from existing taxonomies using Neo4j/` - taxonomy and Neo4j exploration area.
- `Nuclear-project-data-for-use-in-generating-example-frameworks/` - former sample-domain source folder, now empty.
- `Project-framework-by-expert-discussion-and-Yworks/` - expert discussion and Yworks area.
- `image-generation/`, `images/`, and `related work from others/` - support-material locations, now placeholders.

## Rebuilding

Place private or replacement working material outside this public repository unless it is ready to publish. Tiny `.gitkeep` files are present only to keep empty folders visible in GitHub.
